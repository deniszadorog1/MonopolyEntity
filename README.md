MonopolyEntity
A Monopoly board game built with WPF as a learning project to practice C#, Entity Framework, and Material Design in XAML.
Stack

C# / WPF — desktop UI
Entity Framework — data persistence (Code First)
Material Design in XAML — UI components

Getting Started

Clone the repo and open MonopolyEntity.sln in Visual Studio
Restore NuGet packages
Run migrations if needed: (PM> Update-Database)
Hit F5
